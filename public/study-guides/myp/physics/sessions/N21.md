### Q1c (3 marks) — Explain

A student conducts an experiment in which she places a balloon over a glass bottle filled with a gas at room temperature. A Bunsen burner is lit and the balloon expands as the temperature of the gas increases.

**MS answers:**

- (Resulting in) greater pressure [1]

## Key Mark Scheme Patterns

| Pattern | Explanation |
|---|---|
| WTTE | Words To That Effect — accept equivalent phrasing |
| ORA | Or Reverse Argument — accept the logical opposite where applicable |
| Award independently | Both marks can be awarded separately even if one part is wrong |
| Do not award second mark unless first awarded | Marks are dependent — must earn earlier mark to get later one |
| Judge by eye | For drawn responses (graphs, lines of best fit), examiner uses judgment |
| All correct for full marks | Multi-part select/drag tasks — all parts must be correct for the stated marks |
| Unit mark separate | Units are marked independently of the numerical answer |
| Seen or implied | Formula or intermediate step does not need to be explicitly shown |
| Award N marks for correct final answer | If answer is correct, award full marks without requiring working shown |
