## Key Mark Scheme Patterns

| Pattern | Explanation |
|---|---|
| WTTE | Will Take The Equivalent — accept any answer with equivalent meaning |
| ORA | Or Reverse Argument — accept the opposite reasoning where applicable |
| ECF | Error Carried Forward — award marks if subsequent calculation uses a wrong earlier answer correctly |
| Seen or implied | The step does not need to be explicitly written if the answer shows it was used |
| Accept answer to 2 or more sig figs | Unless otherwise stated, answers rounded to 2 sig figs or more are accepted |
| Award unit mark independently | Unit mark is awarded even if the numerical value is wrong |
| Do not accept | Specific wrong answers that are explicitly excluded by the mark scheme |
| [max 1] / [max 2] | Maximum marks available from a bullet-point list even if more correct answers are given |
| Only award 3rd point if first 2 awarded | Conditional marking — higher-level points depend on lower-level points being scored first |
| Markband table | Extended-response questions scored against a 4-column rubric across multiple descriptors |
