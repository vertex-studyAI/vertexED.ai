**Scope**: Mark schemes for all 10 sessions are public; only M24 also has a public question paper. Question stems for N16, M17, M19, N19, N20, M21, N21, M22, M23 are reconstructed from the verbatim quotes in the mark schemes.

**Format evolution**:
- **2016–2021 (N16–N21)**: 7-question papers with sub-parts; mark totals 68–120; mix of short point-marked items (1–4 marks) and extended banded items (8–25 marks).

## 7. Mark-Scheme Patterns (Recurring Rubric Features)

### Verb ladder (the universal IDL spine)
Every session's banded rubrics step through a four-rung verb ladder:

| Band | Verb (typical) |
|---|---|
| Lowest scoring band | states / identifies |
| Lower-mid band | outlines |
| Upper-mid band | describes |
| Top band | explains / justifies / evaluates / develops / compares-and-contrasts |

Confirmed in: N16 (every Q), M17 (Q2c, Q4a, Q6b, Q7), M19 (Q1c, Q3, Q4, Q6, Q7), N19 (every banded Q), N20 (every Q), M21 (every banded Q), N21 (every Q), M22 (every banded Q), M23 (Q1, Q2, Q3), M24 (every Q).

### Best-fit holistic markbands (not point-marking)
Stated explicitly in the rubric of every session. Sample (N16, M17, N20, M21, M22, M23 verbatim):
> "The examiner applying an assessment criterion must choose the achievement level that best matches… It is not necessary for every detailed aspect of an achievement level to be satisfied for that level to be awarded."

Almost all extended tasks use 4 bands (e.g. 0 / 1–3 / 4–6 / 7–9 / 10–12). Short questions use simpler ladders (0/1/2 or 0/1/2/3/4).

### Coverage gates (count of bullet points)
Top-band entry is gated by *covering all bullets* in the question stem.

| Session / Q | Coverage rule |
|---|---|
| N16 Q3a | All 4 sub-bullets required for top band; band 1–3 = no source evidence |
| N16 Q6 | 1 → 2 → 3 → 4 features of sustainability across bands |
| N16 Q7 | 1 → 2 → 3 → 4 reflection bullets across bands |
| M17 Q4a | 1–3 / ≥3 / ≥4 / all 5 bullets across bands |
| M17 Q6a | 1 / 2 / 3 / 4 innovations matched & reasoned |
| N20 Q3a | 2 / 3 / 4 areas correctly used |
| N20 Q3b, Q5 | 1 / 2 / 3 / 4 bullets covered across bands |
| N19 Q5, Q6, Q7 | "at least one / two / three / all" point-coverage gates |
| M19 Q7 | 1 / 2 / 3 / 4 of strengths / limitations / SAA example / conclusion |
| M24 Q4 | 4 bullets; "extensive" reference use at top band |

### Source / evidence gates
Many top bands require a minimum number of source quotations or explicit linkage to pre-release material.

| Session / Q | Source-count or linkage rule |
|---|---|
| N16 Q2b | 1 / 2 / ≥3 examples or quotations across bands |
| N16 Q3a | Band 1–3 = no source; bands 2–4 require source examples |
| M17 Q1c | 0 / ≥1 / ≥2 / ≥3 sources, with outside material at top |
| M17 Q3b | 0 / 1 / ≥2 / ≥3 evidence pieces |
| M19 Q5a, Q5b, Q6, Q7 | Top band requires *explicit linkage* to pre-release material |
| N20 Q1b, Q3a, Q3b, Q6, Q7 | Top band requires "explicit evidence" or "explicitly linked" PRM evidence |
| N21 Q2b | "≥1 / ≥2 / ≥3 pieces of evidence" gate per band |
| N21 Q7 | Top band = "explicitly linked" evidence from the examination |
| M22 Q1b, Q3, Q5, Q6 | Example from Reference 1 mandatory at every band |
| M23 Q3 | "Reference to the delivery driver" alone does not earn the PRM strand — must cite PRM content |
| M24 Q1, Q2, Q3 | Pre-release use ladder: refers to → attempts → uses appropriate → uses *extensive* information |

### Discipline-balance caps
Single-discipline responses are capped at half-marks or below in nearly every paper.

| Session / Q | Discipline cap |
|---|---|
| N16 Q3b | "Maximum of 5 marks if only one discipline mentioned" |
| M17 Q4b | "Maximum of 2 marks if only one discipline addressed" |
| M17 Q5 | Implicit math+I&S balance gate at top band |
| N20 Q3a | "All four areas correctly" required for top band |
| M21 Q4 | "If only one speaker is discussed, then 0 should be awarded" |
| M23 Q2 | Bands 1–6 accept Maths *or* Sciences; bands 7–9 require *and* on one strand; band 10–12 requires *and* on both |
| M24 Q1, Q2, Q3 | Top band requires *both* maths and L&L concepts named explicitly |

### Both-sides / coverage caps
Strength + weakness, for + against, similarities + differences must both be covered to score in upper bands. Stated explicitly in:
- N16 Q1b ("To achieve more than 4 marks both strengths and limitations need to be included")
- M17 Q3b (cap 8/16 if one-sided)
- M19 Q6, M19 Q7 (one-sided answers structurally capped)
- N19 Q1b, Q2c, Q3, Q4, Q6, Q7 (both sides systematically required)
- N20 Q1b, Q7 (both sides required throughout)
- M21 Q6 ("If a candidate only discusses one side, the maximum allowable score is 6 marks")
- N21 Q2b ("if only strengths or only limitations are given, a maximum of 2 marks")
- N21 Q4 ("if only comparisons or only contrasts are given, a maximum of 3 marks")
- M22 Q2c (both strengths and weaknesses required at every band)
- M24 Q4 (positive + negative consequences; one-sided capped at middle band)

### Format / role-play gates
Several questions cap the score if the candidate ignores the requested format or role.

| Session / Q | Format gate |
|---|---|
| N16 Q5 | "To receive 8 marks and above, the context of the script for the social guidelines must be followed" |
| M19 Q5a | Hard-zero if response doesn't address one of the six legal-support areas; canvas-only or text-only capped at 3 |
| N20 Q5 | 6-mark cap if only one city addressed; 3-mark cap if no city chosen |
| M21 Q5a | Role-play frame ("representative for a drug company") must be sustained |
| M22 Q1a | Wrong stakeholder card / wrong dropdown = 0 for that mark |
| N21 Q3a | "Points of view must be in the context of aid distribution. If not, max 3 marks" |
| M24 Q3 | "The structure of the article is not relevant to awarding credit" — synthesis takes precedence over format |

### Conclusion ladder
Top-band entry on extended-response questions hinges on the *quality* of the conclusion:

| Conclusion type | Typical band |
|---|---|
| No conclusion | 1–3 / 1–4 |
| Attempts a conclusion | 3–4 / 4–6 |
| Provides a conclusion | 5–6 / 7–9 |
| Justified / evaluative conclusion | 7–8 / 10–12 (top band only) |

Confirmed in: M17 Q3b, Q6b, Q7; M19 Q6, Q7; N19 Q6, Q7; N20 Q1b, Q3b, Q6, Q7; M21 Q4, Q6, Q7; N21 Q2b, Q6, Q7; M23 Q1, Q2; M24 (implicit).

### Off-task / examiner-blacklist clauses
- M17 Q5: "Do not mark: 'To what extent is it viable to settle on Mars?'" — generic question excluded.
- M21 Q1a: "Use of insulin / treatment of diabetes are not allowed as similarities/differences as they are given in the question."
- M21 Q6: "Responses should relate to whether it should be a right, and not about the pros and cons of it being a right."
- M22 Q2a: "Descriptions of the 'method' or 'workings' to calculate the height is not relevant as this is information provided in the video."
- M23 Q1: "If a response only compares the cars [instead of how the brochures synthesise maths + science], then students score 0 for the first two strands."
- M23 Q2: "Do not credit any scientific information that is not from the examination or PRM."

### Concept-naming requirement (newer rubric feature, M24)
M24 introduced an explicit "concepts must be named, not just used" gate at the top band — students must name *Quantity / Models / Patterns / Representation / Validity* (maths) or *Audience imperatives / Purpose / Style / Point of view* (L&L). Implicit reasoning is read down a band. This is a tightening of the "key concept" gate seen earlier in N16 Q3b (systems), N19 Q2a (global interactions), N21 Q1d (global interactions), M22 Q1b (global interactions).

### "Communicate clearly and coherently → score higher" intra-band lever
A near-verbatim phrase appears in many rubrics: *"Within this markband, answers that communicate the ideas clearly and coherently will score higher."* Used as an internal-band tiebreaker. Confirmed in N16 Q3a, Q3b, Q4b, Q5, Q6, Q7; M17 Q3b, Q4a, Q4b; N20 Q5, Q6.

### Spelling / surface-form leniency
- M24 (verbatim): "Spelling errors should not impact the marks awarded in any part of the examination."
- WTTE / OWTTE clauses appear in N16 Q1c, Q2a; M19 Q1b, Q2d; M22 (throughout examples).
- BOD (Benefit of the Doubt) is an examiner annotation listed in the M22 toolkit.

## 8. Reflective Journal Pattern (Q6 / Q7 / Q4)

Almost every IDL paper closes with one or two metacognitive questions. The shape is highly stable:

| Session | Reflective Q | Marks | Shape |
|---|---|---|---|
| N16 | Q6 (Evaluate); Q7 (Reflect) | 16 + 14 = 30 | Q6: features-of-sustainability coverage gate; Q7: 4-bullet reflection (source / perspectives / questions / classroom prep), 1→2→3→4 across bands |
| M17 | Q7 (To what extent) | 10 | One innovation; PRM evidence + outside-source evidence at top; thorough conclusion |
| M19 | Q6 (To what extent); Q7 (Evaluate) | 12 + 8 = 20 | Q6: 4-bullet coverage; Q7: SAA example required from band 3–4 |
| N19 | Q6 (To what extent); Q7 (Evaluate) | 12 + 8 = 20 | Q6: PRM evidence + evaluative conclusion at top; Q7: examination evidence + evaluative conclusion |
| N20 | Q7 (To what extent) | 8 | Sustainable + not sustainable + PRM evidence + justified conclusion |
| M21 | Q7 (Explain) | 8 | Strengths + weaknesses of own perception + example + evaluative reflection |
| N21 | Q7 (To what extent) | 12 | Local nuclear-plant siting; opinion + disagreement with opposing + examination evidence + justified conclusion |
| M22 | Q6 (Reflect) | 12 | Strengths + weaknesses of own response in Q3/Q4/Q5; example mandatory at every band; PDF-evidence at top |
| M23 | (no separate reflective Q) | — | Reflection embedded inside Q3 |
| M24 | Q4 (Demonstrate) | 6 | Speech for younger students; positive + negative consequences; PRM extensive use at top |

**Universal features of the reflective question:**
1. Bullet-coverage gate (1→2→3→all) determines the band ceiling.
2. Verb ladder states → outlines → describes → explains.
3. Top band requires either an *evaluative* / *justified* conclusion OR explicit reflection on *the candidate's own development of interdisciplinary understanding*.
4. Evidence required from either the PRM or the examination experience (band 3–4 upward).
5. Both-sides requirement on strengths + weaknesses (or for + against, or positive + negative consequences).

## 9. Differences From a Single-Subject MYP Exam

| Feature | Single-subject MYP (e.g. Biology) | MYP Interdisciplinary Learning |
|---|---|---|
| Syllabus | Fixed subject-group syllabus | No fixed syllabus — context-driven each session |
| Disciplines tested | One | Two (and occasionally three) |
| Stimulus | Mostly novel (data, diagrams, scenarios) | Anchored in **pre-release material** issued before the exam — students can prepare |
| Marking model | Mix of point-marked (Crit A & C) and best-fit markbands (Crit B & D) | Almost entirely best-fit holistic markbands; point-marked items reserved for short identifiers |
| Criterion labels in MS | Each question stamped Criterion A/B/C/D | Most IDL MSs do not label individual questions by criterion (M22 is an exception); criteria are implicit in question type |
| Verb ladder | Implicit in command term | Explicit in every band's descriptor — states → outlines → describes → explains is the visible spine |
| Discipline-balance gate | n/a (single discipline) | Pervasive — half-mark caps for single-discipline answers, often hard-cap to 0 in extreme cases (M21 Q4) |
| Source-count gate | Rare | Recurrent — explicit "≥1, ≥2, ≥3 sources/quotations" gates across bands (N16 Q2b, M17 Q1c, M17 Q3b, N21 Q2b) |
| Reflective journal | Embedded in Criterion D essay | A whole question (Q6 / Q7 / Q4) explicitly assesses *metacognition on interdisciplinary understanding* — distinct rubric language |
| Format gates | Rare | Common — script format (N16 Q5), legal-support areas (M19 Q5a), role-play frame (M21 Q5a), correct stakeholder placement (M22 Q1a) |
| Off-task warnings | Occasional | Explicit examiner blacklists (M17 Q5 blacklisted Mars-viability question; M21 Q6 off-task warning; M23 Q1 auto-zero for car-comparison instead of brochure-comparison) |
| Concept-naming | Implicit in scientific terminology | Explicit at top band (M24): "explicit and accurate use of subject concepts" from the related-concepts grids |
| Mark distribution per session | Stable 70–100 | Highly variable: 36 (M23) to 120 (N16) |
| Question count | Usually 6–10 | 3 (M23) to 7 (most sessions) — recent format compression |
| Pre-release material | Not used | Central — questions repeatedly direct candidates to "use evidence from the pre-release material" |
| Synthesis as a graded skill | Not assessed (single subject) | Assessed in its own right in nearly every paper, often as the highest-mark question |
| Service-as-action / personal experience hook | Not present | Sometimes required (M19 Q7 SAA hook from band 3–4 upward) |

**Single most important difference**: in IDL, *covering both disciplines and synthesising them explicitly* is the single most powerful predictor of moving from mid-band to top-band. In a single-subject paper, depth of disciplinary knowledge alone can carry a top-band response.
