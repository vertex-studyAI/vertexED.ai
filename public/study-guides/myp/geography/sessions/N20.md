> - **CONCLUSION**: A global agreement is needed but must set targets for actual greenhouse gas emissions (not just carbon footprint proportions) and include all greenhouse gases to be truly effective

## Key mark scheme patterns (N20)

1. **Specify, don't generalise**: For Q3c (stakeholders), single words like 'students' or 'government' score 0 — must specify "students in a school in Laos" or "Laos government officials"
2. **Source-specific limitations**: For Q6c/6d, the mark scheme rewards responses that identify specific features of the particular graph (e.g. that it only shows one year, that it doesn't show management strategies) — generic limitations of graphs score lower
3. **Outline = state + one elaboration**: For 2-mark "outline" questions throughout, the formula is: name the thing + explain WHY it matters in this specific context
4. **Criterion A (Q5b) + Criterion C**: Q5b and Q8 are assessed across multiple criteria simultaneously. Content marks require depth; communication marks require appropriate register for genre (email vs essay). A geographically accurate response written in poor essay/email style is capped on Criterion C
5. **Action plan evaluation (Q4)**: Full marks for strengths AND limitations AND appraisal — the appraisal must reference a specific aspect of the investigation to score 2 marks; vague overall judgements score only 1
6. **Mitigation over adaptation (Q7)**: The mark scheme example responses consistently use mitigation as the higher-scoring choice because it addresses the root cause — if choosing adaptation, the justification must explain the mechanism, not just describe the strategy
7. **"To what extent" essays (Q8)**: Criterion D (thinking critically) rewards SYNTHESIS across perspectives, not just listing two sides. Upper bands require arguments to build on each other toward a reasoned conclusion
8. **Essay structure for Q8**: C1 Format requires all three structural elements (intro, body, conclusion) as SEPARATE PARAGRAPHS — each missing element loses a mark; paragraphing is explicitly assessed
