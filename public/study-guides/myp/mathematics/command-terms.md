# MYP Standard Mathematics — Command Terms

## What Command Terms Mean for Your Answer

| Command term | What it means | What the mark scheme rewards | What the mark scheme penalises | Example from past papers |
|---|---|---|---|---|
| **Calculate** | Obtain a numerical answer using arithmetic or algebraic working; show every step | Correct method with correct answer; method marks awarded if arithmetic error occurs mid-way | Answer only with no working shown (loses method marks); stopping before rounding when a specific accuracy is stated | M16 Q4c: Calculate the total area of the green shaded segments; N24 Q5c: Calculate the coordinates of the x-intercepts to 1 d.p. |
| **Show that** | Prove a given result using clear, complete steps; the answer is already stated | Every individual step visible (each bullet point in the mark scheme must appear); final line explicitly matches the stated value | Skipping steps; working backwards from the given answer; stating the answer without working | M16 Q4a: Show that the total area enclosed by the circle is 25π; N19 Q8a: Show that the diagonal of the trapezium is 5 units |
| **Describe** | State key features or patterns using words with correct mathematical terminology | Specific correct terminology (e.g. "adds 3 each time", "common difference 8"); more than one valid point for multi-mark questions | Vague language ("it goes up"); incomplete terminology ("arithmetic" without naming the difference); giving the rule in words instead of the pattern | M16 Q1c: Describe fully the rotation that maps shape D to shape A; N24 Q3b: Describe two patterns in the table for S |
| **Find** | Obtain an answer by any correct method; working should be visible | Correct final answer; method marks for correct intermediate steps | Answer only on a multi-step question (loses method marks) | M16 Q3b: Find the coordinates of the point of intersection between L₁ and L₂; N24 Q4e: Find the size of angle FCE |
| **Write down** | State a result that can be read directly from the question, graph, or table — no working required | Correct value; no working needed | None — but a wrong answer scores zero (no method marks available) | M16 Q1a: Write down the equation of the line of reflection; N22 Q2a: Write down the missing values in the two-way table |
| **State** | Give a concise answer, usually a word, phrase, or value; no justification needed | Correct specific term or value | Incorrect terminology; giving extra irrelevant information that contradicts the correct answer | N19 Q3b: State how mutually exclusive events are shown in the Venn diagram; M25 Q3d: State the name given to two events that cannot occur at the same time |
| **Explain** | Give a reason or justification using mathematics or context; must show logical reasoning | A correct reason linked to numerical evidence; appropriate mathematical language | Answer with no reason; reason not linked to numbers or context | M16 Q5d: Did the team score points? Explain your answer; M16 Q9a: Explain why the third term is (1/2)a + (1/2)b |
| **Sketch** | Draw a rough graph or diagram showing key features (intercepts, shape, turning points) without plotting precise values | Correct shape; correct key features labelled (intercepts, asymptotes, max/min) | Missing key features; plotting individual points as if it were a precise graph | (Appears in some sessions as a sub-step within longer questions — reward is for shape and labelled key values, not accuracy) |
| **Plot** | Mark precise points on a grid or draw a graph accurately at given coordinates | Each point placed within tolerance of half a square; line or curve drawn through plotted points | Points drawn as crosses without connecting; line outside tolerance; using a ruler for a curved graph | N22 Q3c: Plot the range of values of x that satisfies the compound inequality on a number line |
| **Justify** | Give evidence or reasoning to support a conclusion; must cite specific values or steps | A correct conclusion supported by a calculation or stated rule; both the claim and its evidence are present | Conclusion without evidence; evidence without an explicit conclusion; "because it works" with no calculation | M16 Q6f: Is the note an A note? Justify your answer; M25 Q3e: Justify whether the simulation result supports the theoretical probability |
| **Suggest** | Propose a reasonable answer in context — an exact numerical answer is not required | Any answer within the acceptable range stated in the mark scheme; must be contextually sensible | An answer outside the allowable range; no answer at all | M21 Q6a: Suggest a sleeping schedule for Ingrid (any 8–10 hour window within the overnight temperature dip) |
| **Determine** | Obtain a specific value or conclusion, usually by calculation or reading from a graph/table; show enough working to make the method clear | Correct value with supporting method; ECF applied if a prior part was used | Answer only with no method shown on a multi-step question | M16 Q2b: Determine the range of shoe sizes required; N22 Q7a: Determine the values of b and c for the linear model |
| **Prove** | Establish a result rigorously using formal logical steps (algebraic or geometric) | Every deductive step present and logically connected; final statement matches what was to be proved | Any gap in the logical chain; using the conclusion as a step in the argument | (Appears in Criterion B investigations as part of the "prove or verify and justify" strand — mark schemes reward each deductive step explicitly) |
| **Verify** | Check that a given general rule or formula is correct by substituting specific values | Substituting a value of n ≥ 5 (a value not given in the original table); explicitly stating that the calculated result matches the predicted value | Only substituting values already given in the table (e.g. n = 1, 2, 3, 4); calculating a result but not confirming it matches the pattern | N19 Q8e: Verify your general rule for S; M23 Q7d: Verify your general rule for T; N24 Q8e: Verify your general rule for S |
| **Generalise** | Express a pattern as a rule that applies to all cases — typically in terms of n | A correct algebraic expression in simplest form with correct notation (e.g. S = 4n, not "four times n"); correct simplification | Rule in words only; using wrong variable (x instead of n) without explanation; rule that does not match all known values | (Underlying intent of "Write down a general rule" questions in Criterion B, e.g. M21 Q8c: Write down a general rule for L; N24 Q8d: Write down a general rule for S) |
| **Investigate** | Explore a mathematical situation systematically: predict, describe patterns, form a general rule, test, verify, justify, and communicate clearly | All strands awarded: Predictions (correct values for n ≥ 5), Description (pattern in words with correct terminology), Rule (correct simplified general rule with correct notation), Testing (substituting n ≤ 4 and confirming match), Verifying (substituting n ≥ 5 and confirming match), Justification (algebraic or geometric proof), Communication (organised and coherent) | Doing only some strands and ignoring others; describing the pattern with vague language; using * / ^ or y instead of n; giving a rule without testing or verifying it; justifying by trying a few values and calling it proof | M16 Q8e: Investigate and find a general rule connecting a, b and L; N22 Q8f: Investigate the total number of squares T in terms of n (20 marks) |
| **Comment** | Make an observation about a result, model, or context — usually about accuracy, validity, or reasonableness | A specific, supported observation (e.g. noting that a model is inaccurate because values may change, or that rounding affects the answer) | Vague statements ("it is inaccurate") with no reason; just restating a calculation result without an observation | N19 Q7e: Comment on the accuracy of the total cost for owning the different vehicles; M23 Q6e: Comment on accuracy of results in the carbon emissions analysis |
| **Outline** | State the main points without detail; similar to Describe but at a higher organisational level | Key points mentioned; not necessarily with supporting calculations | Over-detail that misses the main structure; a single sentence that omits key points | (Used in extended-response criterion D questions requiring students to identify factors before calculating — e.g. identifying factors in M21 Q7f, N22 Q7e) |
| **Identify** | Name or select something specific — a transformation, a statistical measure, a factor, a term | The correct name or item; one item per mark unless otherwise stated | Naming more than one item when one is asked (e.g. "median and IQR" when only one is wanted); vague description instead of a specific term | N22 Q6e–6f: Identify one valid reason why Printer A / B is better; M25 Q4c: Identify the statistical term represented by each shaded quantity |
| **Compare** | State similarities and differences between two or more sets of data or results; must refer to both | Explicit reference to both items using comparison language (higher / lower / same); correct values quoted | Describing only one item without comparison; using vague language ("bigger") without specifying what is bigger or by how much | M16 Q2a: Compare the sizes of shoes sold in Egypt and Brazil; N24 Q3b: Compare the median and IQR for the two distributions |
| **Analyse** | Examine data, models, or a situation in detail; draw reasoned conclusions supported by calculations; typically multi-strand in Criterion D | Multiple strands all present: identifying factors, performing calculations, making a justified recommendation, and commenting on accuracy | Giving a recommendation without supporting calculations; identifying factors but not using them in the analysis; claiming accuracy without a reason | N22 Q7e: Analyse the three face-recognition methods to choose the best; M25 Q7e: Analyse whether to buy Robot 1 or Robot 2 |

## Quick-Reference Danger Zones

| Situation | What goes wrong | Rule |
|---|---|---|
| "Show that" questions | Candidate writes the answer and reverses-engineers the steps | Every forward step must be shown; never start from the target value |
| "Calculate" multi-step questions | Candidate writes only the final answer | Even if the answer is correct, method marks require visible working |
| "Describe" in Criterion B | Candidate writes "it is arithmetic" or "linear" | Name the common difference explicitly: "adding 4 each time" or "common difference 4" |
| "Verify" general rule | Candidate substitutes n = 1, 2, 3 (values already in the table) | Must substitute n ≥ 5 and explicitly state the result matches the predicted value |
| "Investigate" — Justify strand | Candidate tries two extra values and calls this a proof | A geometric or algebraic derivation is needed for full marks; trial-and-verify is a lower band |
| "Compare" | Candidate describes only one distribution | Must refer to both with comparison language and quoted values |
| "State" | Candidate gives an explanation instead of just the term | One precise term is all that is needed — extra information that contradicts the answer loses the mark |
| "Write down a general rule" | Candidate writes the rule in words | The mark scheme explicitly rejects word descriptions; an algebraic expression with correct notation is required |
