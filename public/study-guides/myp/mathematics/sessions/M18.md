## Question 1 — Digit Algorithm & Algebra (7 marks)

**Stimulus:** An algorithm flow diagram operates on a three-digit number X: it reverses X to get X', calculates D = absolute value of (X' − X), reverses D to get D', then outputs D + D'. A simulator demonstrates the algorithm with any chosen three-digit number.

### Q1a (3 marks) — Apply
Apply the same algorithm to 437 to complete the missing values.

[Flow diagram: X = 437 → X' = ? → D = ? → D' = ? → D + D' = ?]

**Mark scheme:**
- •¹ 437 and their D correctly reversed → **734 and 792**
- •² their D correct → **(their 734 − 437 =) 297**
