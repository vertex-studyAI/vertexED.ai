### Question 2 — Homeostasis & Transport / Dialysis (11 marks)

**Context:** The human body depends on transport processes to maintain homeostasis. The video demonstrates how transport processes are used to treat medical conditions. The kidneys filter the blood, removing excess water, toxins and waste products; they also regulate mineral levels and blood pressure, make vitamin D, and maintain pH balance. The kidneys filter ~300 times the body's blood per day. Dialysis is used when kidneys fail — blood passes through a dialyzer (artificial kidney) containing fine fibres with pores; the process uses diffusion and osmosis. Sessions last up to six hours, repeated up to three times per week.

## Key Mark Scheme Patterns

| Pattern | Description |
|---|---|
| **WTTE** | "Will take the equivalent" — accept answers expressing the same idea in different words |
| **ORA** | "Or reverse argument" — accept the opposite valid argument (e.g. if sugar decreases, heart rate decreases) |
| **ECF** | "Error carried forward" — if a student uses a wrong answer from a previous part correctly in a subsequent calculation, award the method marks |
| **CON annotation** | If a student duplicates variables (e.g. states same variable as both IV and CV), do not award marks for the duplicated variable; add CON annotation |
| **Criterion A** | Knowing and understanding (factual recall, definitions, concepts) |
| **Criterion B** | Inquiring and designing (variables, hypothesis, method design) |
| **Criterion C** | Processing and evaluating (calculations, graph work, evaluation of data, weaknesses) |
| **Criterion D** | Reflecting on impacts of science (extended essays, ethical/social/environmental reasoning) |
| **Markband questions** (Q5, Q8) | Assessed holistically across 5 strands; score each strand 1–4 then sum for total out of 17 |
| **"Fourth mark" bonus** (Q1d) | Scientific terminology must be used correctly to earn the fourth mark in the natural selection explanation |
| **Do not accept "water concentration"** | For osmosis, only "water potential" is accepted — not "water concentration" |
| **At least 5 IV levels** | For any investigation design, minimum of 5 levels of the independent variable needed to show a trend |
| **Minimum 3 repeats** | For reliability; justification must link to reducing random error, calculating mean, or identifying anomalies |
