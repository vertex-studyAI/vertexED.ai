## Key Mark Scheme Patterns

| Pattern | Description |
|---|---|
| **WTTE** | "Will To The Effect" — accept synonymous language/wording |
| **ECF** | Error Carried Forward — accept consequent answers from earlier errors |
| **All-or-nothing** | Some drag-and-drop questions (Q1a) only award marks if all items are fully correct |
| **Separate marks** | Drag-and-drop with zones (Q1b): marks awarded per zone or per item |
| **Two marks, two examples** | When asked for two, do not award two marks for two examples of the same type |
| **Named example required** | Some questions (Q7a) require a specific named contaminant, not just generic "contaminant" |
| **Change must be stated** | Q8c: change must be clearly stated, not just implied |
| **Markband criteria** | Q5, Q9b: extended response uses a markband table across multiple sub-criteria |
| **Do not accept** | GMOs not good for human health (Q9a); bigger nets alone (Q8b); general refs to CV as limitation (Q4g); exchange of materials for mitochondria (Q1d); storage of energy for mitochondria (Q1c) |
