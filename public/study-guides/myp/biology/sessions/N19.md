## Key Mark Scheme Patterns

| Pattern | Detail |
|---|---|
| WTTE | "Words to that effect" — accept equivalent phrasing throughout |
| Criterion allocation | A = knowledge/understanding; B = inquiry/design; C = processing/evaluation; D = reflection/communication |
| "max" notation | e.g. "any two from the list (2 max)" — only the number of marks stated are awarded regardless of extra correct answers |
| Accept / Do not accept | Explicit inclusions and exclusions noted; e.g. "Accept tropic web / Do not accept food chain" |
| Implied answers | For design questions, variables and DV may be implied within the method |
| Significant figures | Q5g requires 2 sig figs; Q3a DV requires indication of quantity |
| Linked justification | Improvements (Q3c, Q5i) must be paired with a correctly linked justification to earn both marks |
| WTTE on trend questions | Q5e: accept trends described in explanation box only; ignore "exponential" misuse |
| Criterion D terminology | Q2d requires correct use of "translocation" or "phloem"; Q4b requires "active site" or "lock-and-key" |
