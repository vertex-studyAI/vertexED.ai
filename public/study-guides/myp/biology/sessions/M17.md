## Key Mark Scheme Patterns

| Pattern | Detail |
|---|---|
| **WTTE** | "Words to that effect" — accept equivalent phrasing throughout |
| **ORA** | "Or reverse argument" — accept converse statement |
| **Contradiction rule** | Where advantage AND disadvantage are both required, they must not contradict each other (Q2d, Q2e) |
| **Justification linked** | Justification marks are only awarded if the preceding claim is scientifically valid (Q5c, Q6a, Q6d) |
| **Do not accept "tree" alone** | Q8b requires "Japanese/foreign/non-native/exotic" as qualifier |
| **Do not accept "use better equipment"** | Q5c — improvement must be methodological/procedural |
| **Do not accept reference to control variable** | Q4b, Q4c — research question and weakness questions exclude CVs |
| **3 sig fig rule** | Q6d — calculation must be given to 3 significant figures |
| **Markband questions** | Q7 (21 marks, Criterion B) and Q8f (15 marks, Criterion D) use holistic markbands, not point-by-point marking |
| **Criterion codes** | A = Knowing & Understanding; B = Inquiring & Designing; C = Processing & Evaluating; D = Reflecting on the Impacts |
