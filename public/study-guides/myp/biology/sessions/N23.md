## Key Mark Scheme Patterns

| Pattern | Detail |
|---|---|
| **WTTE** | "Words To That Effect" — accept synonymous phrasing for answers not requiring precise terminology |
| **ORA** | "Or Reverse Argument" — accept the converse of the model answer where applicable (e.g. Q3d hypothesis) |
| **ECF** | "Error Carried Forward" — if a student makes an error in an earlier part, their answer to a later part is marked on the basis of their earlier (incorrect) answer, provided method is correct |
| **Accept any reasonable...** | Open-ended questions (e.g. Q1g, Q1h, Q2f, Q3f) reward any scientifically valid response, not only listed examples |
| **Linked marks** | In Q2e and Q3f, the second mark of a pair can only be awarded if the first mark of the pair is already awarded |
| **Units required** | Q4b rate answer, Q4c axis labels — units must be present for full marks |
| **No named examples** | Q2c homeostasis definition must be general; do not accept named examples (e.g. "blood glucose stays at X") |
| **Criterion levels** | Q3c (Criterion B), Q5 (Criterion B, 19 marks), Q6c (Criterion D), Q7a (Criterion D, 14 marks) use markband rubrics rather than point-scoring |
| **Do not accept** | Q3d: Do not accept "lack of oxygen" as reason CO₂ unchanged; Q6a: Do not accept "use of fences"; Q7b: Do not accept "vaccine may cause harm" |
| **Positive correlation phrasing** | Q4g: accept "positive correlation" for the interpret question about UV exposure and DNA changes |
