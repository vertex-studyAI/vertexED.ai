## Key Mark Scheme Patterns

| Pattern | Details |
|---|---|
| **WTTE** ("Words to that effect") | Used extensively — paraphrases are acceptable if the biological meaning is correct |
| **CON** (Contradiction) | If a student contradicts a correct answer with an incorrect statement, the mark is not awarded (e.g. Q1d: mentioning cell wall after correctly identifying cell membrane function) |
| **ECF** (Error carried forward) | Applied in Q6d — if an earlier calculation error is carried forward correctly, marks are still awarded for subsequent working |
| **Criterion separation** | Questions 1–3 = Criterion A; Questions 4 (parts) & 6 = Criteria B & C; Questions 5 = Criterion B; Questions 7–8 = Criterion D |
| **"Do not accept"** | Specific exclusions are frequent: e.g. "easier" for digestion (Q2c), "colour/shape/vacuole" for plant/animal distinction (Q1b), "accuracy" vs "precision" (Q4e/4f), "aquatic plant" for pondweed explanation (Q6f) |
| **Markbands for extended tasks** | Q5 (Design, 17 marks) and Q8a (Discuss & Evaluate, 13 marks) use multi-criterion markband tables, not point-by-point mark schemes |
| **Terminology mark** | Q7b awards 1 mark explicitly for correct use of enzyme terminology (catalyse, active site, substrate, induced fit, lock-and-key, etc.) |
| **Named examples** | Generally accepted as WTTE in knowledge questions (e.g. Q1e accepts named cells/tissues) |
| **"Accept disaccharide or monosaccharide"** | Q2d — both levels of sugar breakdown accepted |
| **Sequential marking** | Q3c: third mark only awarded if second mark is awarded first (chain-of-consequence logic) |
| **Significant figures** | Q6d: answer must be rounded to 89 mm⁻² (2 s.f.) for full marks; ECF rules apply across sub-parts |
