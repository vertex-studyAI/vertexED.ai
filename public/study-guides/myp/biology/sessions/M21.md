## Key Mark Scheme Patterns

| Pattern | Description |
|---|---|
| **ORA** | Or reverse argument — accept the logically opposite reasoning if consistently applied |
| **WTTE** | Words to that effect — accept any phrasing with the same meaning |
| **ECF** | Error carried forward — if a wrong answer in an earlier part is used consistently in a later part, award marks |
| **[x max]** | Only award up to x marks from that bullet list |
| **"or"** | Alternative acceptable answers; only one needed |
| **"and"** | Both elements required for the mark |
| Criterion letters | A = Knowledge & Understanding; B = Inquiring & Designing; C = Processing & Evaluating; D = Reflecting on the impacts of science |
| Markband questions | Q6 (Design, 15 marks, Crit B) and Q9 (Discuss/Evaluate, 11 marks, Crit D) and Q10b (6 marks, Crit D) use holistic markbands, not point-by-point marks |
| "Do not accept" | Explicit rejections noted for common misconceptions (e.g. do not accept "plant" for producer; do not accept "clearer" for 3D model advantage) |
