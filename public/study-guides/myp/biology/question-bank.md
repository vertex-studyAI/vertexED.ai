## Section 4 — Filter by Command Term

### State
Short recall questions requiring a specific fact, name, or value. Single mark or low-mark sub-parts. Used in all sessions across all criteria.

Common examples:
- **M16 Q1b** (1m) — State the name of the organ system that carries blood
- **M16 Q2a** (1m) — State the name of the genetic molecule
- **M16 Q8a** (1m) — State the definition of biodiversity
- **M16 Q9a** (1m) — State the name of the diagram type shown
- **M16 Q10a** (1m) — State one reason seed banks are important
- Every session has multiple "State" sub-questions — typically 1–2 marks each

### Outline
Brief description requiring 2–3 key points. More than "state" but less than "explain." Typically 2–4 marks.

Common examples:
- **M16 Q1c** (2m) — Outline why oxygen uptake requires no energy
- **M16 Q2b** (3m) — Outline events during mitosis anaphase
- **M16 Q2c** (3m) — Outline three differences between mitosis and meiosis
- **M16 Q10b** (3m) — Outline the process to create a seed bank
- **N18 Q2** (partial) — Outline the immune response to vaccination
- Every session has multiple "Outline" sub-questions

### Explain
Cause-and-effect or mechanism questions. Requires reasons or mechanisms to be given. Typically 2–4 marks.

Common examples:
- **M16 Q1e** (4m) — Explain senses and survival responses
- **M16 Q2e** (3m) — Explain how meiosis causes genetic differences
- **M16 Q6b** (3m) — Explain scientific reason for difference in biomass groups
- **M16 Q8c** (4m) — Explain how loss of one species affects ecosystem organisms
- **N18 Q4** (partial) — Explain why cricket chirp rate varies with temperature

### Describe
Give a detailed account of a structure, process, or observed pattern. Typically 2–4 marks.

Common examples:
- **M16 Q2d** (2m) — Describe problems to genetic molecule during meiosis
- **M16 Q6d** (4m) — Describe a strength and weakness of the investigation method
