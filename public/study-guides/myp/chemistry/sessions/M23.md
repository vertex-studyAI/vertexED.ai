## Mark scheme conventions used

| Convention | Where used | Meaning |
|------------|-----------|---------|
| **WTTE** | Q1d, Q3a (implied), Q4e, Q5e | Words to that effect — any phrasing conveying the same meaning is accepted |
| **ORA** | Q1d, Q2b (implied), Q3a, Q3d, Q4g, Q5e | Or reasonable alternative — sensible equivalent answers accepted |
| **ECF** | Q3c, Q4b | Error carried forward — follow-through marks awarded if earlier error is consistent |
| **CON** | Q1c | Contradiction — if student contradicts themselves, zero marks awarded (e.g. both molecules named as alcohols) |
| **Do not accept** | Q4b, Q4g, Q5a, Q5b, Q5d | Explicit exclusions for common wrong answers |
